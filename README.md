## 项目地址
```
git clone https://github.com/zhangfeng001/react-webpack.git
```

## 安装依赖
```
npm install 
```

## 启动本地服务器
```
npm run start
```

## 打包发布

```
npm run build

```
## 项目搭建方法

- 使用webpack搭建，非脚手架，
